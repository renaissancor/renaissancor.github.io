## 소개

**C/C++**와 **시스템 수준 프로그래밍**에 탄탄한 기반을 두고, **웹 개발**·**시스템 프로그래밍**·**ML/데이터 인프라**를 아우르는 유연한 **소프트웨어 엔지니어**.

온프레미스 **H200 GPU** 인프라 위에서 **VLM·OCR·ASR** 추출 파이프라인으로
**수백만 건의 자산**을 처리하고 있습니다.

이전에는 **IOCP 기반 비동기 I/O**, **멀티스레드 아키텍처**, **Windows 시스템 프로그래밍** 등
네트워크·서버 프로그래밍을 깊이 탐구했습니다.
다양한 국제 환경에서 기술/비기술 팀을 연결하며,
효율적이고 확장 가능한 솔루션을 전달하는 데 열정을 가지고 있습니다.

## 경력

### **i-Scream Media**  
_2026년 2월 – 현재_  
**주니어 연구원**, 대한민국 서울

- 대규모 어린이 교육 미디어 라이브러리에서 메타데이터·커리큘럼 태그를 추출하는 **온프레미스 멀티모달 콘텐츠 이해 플랫폼**을 구축·확장. **이미지·영상·문서·인터랙티브 HTML·레거시 Flash** 자산을 통합 메타데이터 계약 아래 처리.
- **H200 GPU** 위에서 **vLLM**으로 **VLM 이미지 안전성 분류기**(**Gemma 4 31B, FP8**)를 개발 — VLM 시각 분석 + 결정론적 비속어/욕설 텍스트 분석의 **2-패스 파이프라인**으로 한국 미디어 등급(전체/12/15/18)을 산출하고 **FastAPI** 프로덕션 서버로 서빙.
- 트랜잭션 클레임 큐·체크포인트·워치독 기반 복구를 활용한 **고처리량·재개 가능 추출 파이프라인**을 설계하여 **이미지 500만+ 건, 커리큘럼 150만+ 레코드를 99.9%+ 성공률**로 처리. 배치 최적화로 수주 예상 작업을 수일로 단축(**약 5배 처리량**).
- **영상·문서 AI 파이프라인** 구축 — **Whisper** 전사·**VAD** 기반 프레임/오디오 추출, PDF/PPT/HWP 대상 레이아웃 인식 OCR(**PaddleOCR-VL**).
- 플랫폼 **데이터 계층** 설계: 안정적 콘텐츠 해시 식별자, 콘텐츠별 스키마, 운영용 **MySQL / PostgreSQL** 대비 **50–180배 빠른** 분석용 **DuckDB / Parquet** 마트.
- **Mac → H200 배포 자동화**(독립형 rsync 스크립트, systemd 서비스, 잠금된 `uv` 환경) 및 커버리지·데이터 품질 검수를 위한 **Streamlit / 정적 QA 대시보드** 구축.
- 어린이 데이터 프라이버시 제약 아래 비용 효율적 추론을 위해 **FP8/AWQ 양자화**·텐서 병렬 토폴로지·동시성 튜닝으로 공용 멀티 GPU 인프라(**H200 NVL**, **RTX 4090**) 운영.
